# 📊 Análise de Dados de Vendas com Python

Análise exploratória de dados de uma loja online com mais de 51.000 registros usando Python.

## 📁 Sobre o Dataset
O dataset contém informações sobre pedidos de uma loja online, incluindo:
- Data e hora do pedido
- Informações do cliente (gênero, tipo de login, dispositivo)
- Detalhes do produto (categoria, nome)
- Dados financeiros (vendas, lucro, desconto, custo de envio)

## 📈 Visualizações
4 tipos de gráficos criados com Matplotlib:

- **Histograma** — distribuição dos preços de venda
- **Gráfico de Linha** — tendência de vendas ao longo do tempo (2018)
- **Gráfico de Barras** — número de pedidos por categoria de produto
- **Gráfico de Dispersão** — correlação entre lucro e vendas

## 🔍 Principais Insights
- Os preços de venda se concentram em duas faixas: $110-120 e $220-230
- As vendas cresceram significativamente a partir de março/2018
- **Fashion** é a categoria mais popular (~25.000 pedidos)
- **Eletrônicos** é a menos popular (~2.700 pedidos)
- Há uma correlação positiva forte entre lucro e vendas
